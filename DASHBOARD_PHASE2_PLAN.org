* Database Schema

** Settings Table

#+BEGIN_SRC sql
CREATE TABLE IF NOT EXISTS suites (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    folder_path TEXT UNIQUE NOT NULL,
    suite_name TEXT NOT NULL,
    mode TEXT DEFAULT 'docker' CHECK(mode IN ('standalone', 'docker')),
    config_json TEXT,              -- cached tsuite.yaml as JSON
    test_count INTEGER DEFAULT 0,  -- number of discovered tests
    last_synced_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_suites_folder_path ON suites(folder_path);
#+END_SRC

** Fields Description

| Field          | Type      | Description                                    |
|----------------+-----------+------------------------------------------------|
| folder_path    | TEXT      | Absolute path to suite folder (unique key)     |
| suite_name     | TEXT      | Display name (from yaml or folder name)        |
| mode           | TEXT      | 'standalone' or 'docker'                       |
| config_json    | TEXT      | Full tsuite.yaml cached as JSON                |
| test_count     | INTEGER   | Number of test cases discovered                |
| last_synced_at | TIMESTAMP | When config was last read from yaml            |
| created_at     | TIMESTAMP | When suite was first added                     |
| updated_at     | TIMESTAMP | Last modification time                         |

* API Specification

** Suite Management

*** GET /api/suites
List all registered test suites.

#+BEGIN_SRC json
{
  "suites": [
    {
      "id": 1,
      "folder_path": "/Users/dev/mcp-mesh-test-suite/integration",
      "suite_name": "MCP Mesh Integration Tests",
      "mode": "docker",
      "test_count": 29,
      "last_synced_at": "2026-01-19T10:00:00Z",
      "created_at": "2026-01-19T09:00:00Z"
    }
  ],
  "count": 1
}
#+END_SRC

*** POST /api/suites
Add a new test suite by folder path.

Request:
#+BEGIN_SRC json
{
  "folder_path": "/path/to/test/suite",
  "mode": "docker"
}
#+END_SRC

Response:
#+BEGIN_SRC json
{
  "id": 2,
  "folder_path": "/path/to/test/suite",
  "suite_name": "My Test Suite",
  "mode": "docker",
  "test_count": 15,
  "tests": [
    {
      "use_case": "uc01_setup",
      "test_case": "tc01_install",
      "name": "Install CLI",
      "tags": ["setup", "smoke"]
    }
  ]
}
#+END_SRC

Errors:
- 400: Invalid path or not a valid tsuite folder
- 409: Suite already exists (folder_path conflict)

  "mode": "standalone",

*** DELETE /api/suites/:id
Remove suite from settings (does not delete files).

*** POST /api/suites/:id/sync
Re-read tsuite.yaml and update cached config.

Response:
#+BEGIN_SRC json
{
  "synced": true,
  "test_count": 29,
  "last_synced_at": "2026-01-19T12:00:00Z"
}
#+END_SRC

** Test Discovery

*** GET /api/suites/:id/tests
List all tests in a suite, organized by use case.

Query params:
- =uc= - Filter by use case (e.g., =uc01_registry=)
- =tag= - Filter by tag (e.g., =smoke=)

#+BEGIN_SRC json
{
  "suite_id": 1,
  "tests": [
    {
      "test_id": "uc00_setup/tc01_install_meshctl",
      "use_case": "uc00_setup",
      "test_case": "tc01_install_meshctl",
      "name": "Install meshctl CLI",
      "description": "Downloads and installs the meshctl CLI",
      "tags": ["setup", "smoke", "install"],
      "steps_count": 5,
      "estimated_duration_ms": 6400
    }
  ],
  "count": 29
}
#+END_SRC

** Test Execution

*** POST /api/suites/:id/run
Start a test run for a suite.

Request:
#+BEGIN_SRC json
{
  "tests": ["uc01_registry/tc01_agent_registration"],  // optional, runs all if empty
  "tags": ["smoke"],                                    // optional, filter by tags
  "mode": "docker"                                      // optional, override suite mode
}
#+END_SRC

Response:
#+BEGIN_SRC json
{
  "run_id": "abc-123-def",
  "status": "started",
  "tests_to_run": 5,
  "stream_url": "/api/runs/abc-123-def/stream"
}
#+END_SRC

*** GET /api/runs/:id/cancel
Cancel a running test (future).

** Sync on Run

When tsuite runs from CLI, sync suite config to DB:

#+BEGIN_SRC python
# In cli.py, before running tests
def sync_suite_to_db(folder_path: str, config: dict):
    """Upsert suite config to settings table."""
    suite = Suite(
        folder_path=os.path.abspath(folder_path),
        suite_name=config.get('name', os.path.basename(folder_path)),
        mode='docker' if args.docker else 'standalone',
        config_json=json.dumps(config),
        test_count=len(discovered_tests),
        last_synced_at=datetime.utcnow()
    )
    repo.upsert_suite(suite)
